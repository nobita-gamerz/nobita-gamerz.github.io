INSTALL UBUNTU IN TERMUX

   Linux merupakan System Operasi untuk Komputer (Bahasa Sederhana nya). Tapi dengan Aplikasi Termux anda bisa menjalankan berbagai distro linux di ponsel android dengan mudah.


    Kakashi kali ini ingin menulis cara install berbagai distro linux di android menggunakan PRoot.Tujuan utama PRoot adalah untuk menjalankan distribusi Linux di dalam Termux tanpa harus melakukan root pada perangkat, jadi anda bisa menjalankan program linux yang membutuhkan akses root.


    Pertama kalian harus download terlbih dahulu termux di play store,kemuadian setalh kalian download termux,kalian open app termuxnya masukan perintah dibawah ini:

    1. ketikan perintah apt update && upgrade (bila terjadi error)
    - apt install root-repo
    - apt install unstable-repo
    - apt install x11-repo
    2. setelah kalian install pakage di atas kalian bisa langsung masukan perintah di bawah ini :
    - apt install proot
    - apt install proot-distro (tunggu hingga selesai)
    - proot-distro list (untuk memilih list distro)
    - proot-distro install ubuntu (tunggu hingga selesai)
    - proot-distro login ubuntu
    3. lakukan update dan upgrade (apt update && apt upgrade) (bila terjadi error)
    - nano /etc/resolv.conf
    - gantilah name server 8.8.8.8 dan name server 8.8.4.4 bila sudah ctrl+x lalu y (untuk menutup aplikasi nano)
